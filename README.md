# Homebrew Tap

Homebrew tap for [MAUI Sherpa](https://github.com/Redth/MAUI.Sherpa) and other tools.

## Installation

```bash
brew install --cask redth/tap/maui-sherpa
```

Or manually tap first:

```bash
brew tap redth/tap
brew install --cask maui-sherpa
```

## Updating

```bash
brew upgrade --cask maui-sherpa
```

## Available Casks

| Cask | Description |
|------|-------------|
| `maui-sherpa` | Desktop app for managing .NET MAUI developer tools |
